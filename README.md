Software for Vocational Training for Persons with Visual Impairment
🔹 Overview

This project is an accessibility-first web/mobile application designed to provide vocational training for visually impaired individuals. The platform removes physical and linguistic barriers by offering voice-driven navigation, audio-based learning, and real-time interaction, enabling users to become more independent and job-ready.

🎯 Problem Statement

Visually impaired individuals often face:

Limited access to offline training due to mobility constraints
Lack of accessible digital platforms
Communication barriers with mainstream learning systems

This solution bridges that gap through an inclusive, voice-powered learning ecosystem.

1. Frontend (Flutter App)

Provides accessible UI with screen reader compatibility
Handles voice commands and navigation
Offline-first support using PWA concepts

2. Voice Processing Layer

Converts speech → text (commands)
Converts text → speech (responses)
Ensures hands-free interaction

3. Backend (Node.js + Express)

Manages business logic
Handles API requests (courses, tests, chat, etc.)
Connects frontend with Firebase services

4. Firebase Services

🔐 Authentication (user login/signup)
📂 Realtime Database (user progress, chat, content)
☁️ Storage (audio courses, media files)

5. Google Cloud

Scalable hosting environment
Supports AI-based enhancements (future scope)
Ensures reliability and performance
